# YouTube Video Card Web App



## Description
This is a simple web application that generates a YouTube video card for sharing purposes. It allows users to input a YouTube video URL and creates a card displaying basic video information like the title, thumbnail, and channel name.

## Features
- **Video Preview:** Automatically pulls the thumbnail and title from the YouTube URL.
- **Card Display:** Clean and responsive video card layout.
- **Easy Sharing:** Simple format for copying or sharing video cards.

## Demo
You can view the live demo of the application here: [YouTube Video Card Demo](https://youtubevideocard.netlify.app/)
