## Project Overview

This project is a **personal resume website** that highlights:

- **Personal Info:** Name, profession, and contact details  
- **About Me:** Short professional introduction  
- **Education:** Academic qualifications displayed in a table (mobile-friendly)  
- **Skills:** Programming languages, tools, and other relevant skills  
- **Certificates:** Professional and educational achievements  
- **Languages:** Known languages displayed horizontally  
- **Projects:** Major projects with description and links  

The website is **fully responsive**, readable on **desktop and mobile**, and designed to be **print-friendly for PDF export**.

---

## Features

- **Clean and professional design**  
- **Mobile-friendly and responsive layout**  
- **Horizontal layout for languages**  
- **Stacked and readable project cards**  
- **Subtle highlight for skills, certificates, and projects**  
- **Responsive education table for mobile devices**  
- **Contact links** (Email, Phone, LinkedIn, GitHub)
